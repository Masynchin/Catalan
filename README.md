# Catalan

Naive way for generating Catalan numbers sequence in Haskell

## How to run

Clone repo with `git`:

~~~shell
https://github.com/Masynchin/Catalan.git
cd Catalan
~~~

Compile and run with `ghc`:

~~~shell
ghc main.hs
./main
~~~

...or in Docker:

~~~shell
docker run --rm -it $(docker build -q .)
~~~
